## Prerequisites 

Install [genash-cli](https://github.com/trufflesuite/ganache-cli):

Using npm:

```Bash
npm install -g ganache-cli
```

or, if you are using [Yarn](https://yarnpkg.com/):

```Bash
yarn global add ganache-cli
```
